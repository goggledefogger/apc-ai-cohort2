## Before Session 1: Pre-Flight Setup

Complete this before our first class. Everything you need — accounts, tools, and verification — is in the guide:

**[Pre-Flight Setup Guide](https://goggledefogger.github.io/apc-ai-cohort2/docs/pre-flight-setup-guide.html)**

The guide walks you through:
1. Creating your accounts (GitHub, Claude Pro, Cursor)
2. Installing Cursor and opening your first folder
3. Letting Cursor install your dev tools (Node.js, Git, GitHub CLI, Claude Code)
4. Launching Claude Code for the first time
5. Setting up Handy (voice dictation)
6. Running the pre-flight verification check

If you get stuck on any step, don't spiral — just note where you stopped and let us know. We'll sort it out before class.

## Claude Code Skills

These are helper files that teach Claude Code how to guide you through specific tasks. The pre-flight guide will walk you through installing them, but if you need to do it manually:

```bash
mkdir -p ~/.claude/commands
curl -o ~/.claude/commands/course-preflight.md https://raw.githubusercontent.com/goggledefogger/apc-ai-cohort2/main/skills/course-preflight.md
curl -o ~/.claude/commands/preflight-check.sh https://raw.githubusercontent.com/goggledefogger/apc-ai-cohort2/main/skills/preflight-check.sh
curl -o ~/.claude/commands/handy-setup.md https://raw.githubusercontent.com/goggledefogger/apc-ai-cohort2/main/skills/handy-setup.md
```

Then in Claude Code, run `/course-preflight` to verify your setup.
